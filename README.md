# Euler cycle visualization

Working in progress project...

## Requirements

- Nodejs
- Rustup, Visual Studio Build Tools (or GNU)

## Development

1. Clone project

```bash
git clone https://github.com/nghiaxh/euler-cycle.git
```

2. Install dependencies

```bash
npm install    # or yarn install
```

3. Run project

```bash
npm run tauri dev   # or yarn tauri dev
```
